# Actor: AIS - Monitoramento de Datas

Este ator é responsável pela vigilância contínua da disponibilidade de vagas nos postos consulares.

**URL**: [ais.usvisa-info.com](https://ais.usvisa-info.com/)

## Responsabilidades
- **Polling de Calendário**: Verificação sistemática de slots abertos para CASV e Entrevista.
- **Log de Disponibilidade**: Registro histórico de janelas abertas para fins estatísticos e de estratégia de antecipação.
- **Alertas**: Notificação ao sistema quando uma data desejada (conforme requisitos da assessoria) é detectada.
